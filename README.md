# FocusNow - Aplicación Web de Productividad

Aplicación web full stack desarrollada con React y Node.js.

## Tecnologías utilizadas
- React.js
- Node.js
- Express
- CSS3

## Funcionalidades
- Interfaz responsiva
- Consejos dinámicos según perfil
- Rutas HTTP con Express
- Integración Front-End y Back-End
